Image Generator in Python

Project idea was born when friend was developing trading discord bot for game called "Dark and Darker".
Idea is that based on information passed into function, file generates Image.
Using Pillow library to generate image. Image height is dynamic based on how many "modifiers" are for item.
It moves text lover to be able to insert required extra text fields in middle.
Top part of Image is being updated with coresponding colour for item Rarity
Icon images are being passed into function as variable. Image is being resized and applied on top of background image.

Lingo in C#

Simple Lingo game created in C#.
Application chooses random 5 letter word from pre-definied list.
Correct location letters are marked in green.
If letter is in wrong location, but still is in chosen word, it's marked in light blue.
If letter is not in word, it's simply left with white background.

Cost Calculator C#

Application to calculate output based on provided data.

  1. Line physical speed (meters per second)
  2. Spacing (in which position product is placed)
  3. Job Type
  4. Target %
  5. Hourly rate

Calculator is calculating output based on provided information. It gives cost per unit.
Calculation is done based on physical distance between "cups/spaces and actual line speed."
